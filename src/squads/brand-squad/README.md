# Brand Squad

The definitive brand strategy squad — 10 legendary brand thinkers + 4 functional specialists + 1 orchestrator.

## Quick Start

```
@brand-chief        # Activate the orchestrator
*diagnose            # Triage your brand challenge
*brand-creation      # End-to-end brand creation workflow
*rebrand             # Rebrand an existing brand
```

## Components

- **15 agents** — 1 orchestrator + 10 thinkers + 4 specialists
- **9 tasks** — audit-brand, diagnose, review, create-positioning, generate-names, build-identity, create-brand-story, design-architecture, map-archetype
- **2 workflows** — brand-creation (full), rebrand (existing brands)
- **1 checklist** — output-quality
- **2 data files** — brand-frameworks, routing-catalog

## Workflows

### Brand Creation (`*brand-creation`)
End-to-end: archetype > positioning > naming > identity > story > launch package.

### Rebrand (`*rebrand`)
Audit > positioning review > identity update > messaging refresh.
