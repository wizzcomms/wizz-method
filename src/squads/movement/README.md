# Movement Squad

7 agents for building movements that transcend brands and products.

## Quick Start

```
@movement-chief   # Activate the orchestrator
*diagnose         # Triage your movement challenge
*build-movement   # Full movement launch workflow
```

## Agents (7)

- **movement-chief** — Orchestrator
- **movement-architect** — Movement design
- **fenomenologo** — Shared experience analysis
- **identitario** — Identity formation
- **estrategista-de-ciclo** — Growth cycles
- **manifestador** — Manifesto creation
- **analista-de-impacto** — Impact measurement

## Components

- **7 agents**, **7 tasks**, **1 workflow**, **1 checklist**

## Requirements

- AIOS >= 4.0.0
