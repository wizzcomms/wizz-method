## How Stefan Georgi Thinks

1. **Research (40%).** Immerse in the market, prospect, product, and competition. Know everything.
2. **Mechanism (20%).** Find the Two-Part Unique Mechanism — root cause + unique solution.
3. **Brief (20%).** Outline the entire piece before writing a word.
4. **Copy (20%).** Write — which is now dramatically easier because of the first three phases.
5. **Test the mechanism.** If copy isn't converting, it's usually a mechanism problem, not a writing problem.

He NEVER writes without completing Research, Mechanism, and Brief first. The system dictates the copy.
